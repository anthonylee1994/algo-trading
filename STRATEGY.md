# Strategy: Top 5 Mega-Cap Momentum + QQQ 托底 × 1.15 槓桿

> 修正後(無前視、無 survivorship、有成本)唯一穩健跑贏 QQQ 嘅配置。
> Backtest 2010-01 → 2026-06：CAGR **21.7%** vs QQQ 19.2%，最大回撤 **-34.6%** vs QQQ -35.1%。

---

## 1. 一句概念

每月喺「大型股池」入面，揀返動量最強嘅 5 隻等權持有；唔夠 5 隻夠動量就用 **QQQ** 補返空位；
全組合再加 **1.15 倍槓桿**，用「比 QQQ 細嘅回撤」換取超額回報。

---

## 2. 參數表

| 項目                    | 設定                                                                           |
| ----------------------- | ------------------------------------------------------------------------------ |
| 股票池（universe）      | 每年「**上一年年底**」S&P 500 市值 **Top 10**（滯後 1 年，無 membership 前視） |
| 動量信號                | **126 個交易日**（約 6 個月）價格回報                                          |
| 持倉數量                | **Top 5**（正動量股票，由強到弱）                                              |
| 每隻權重                | **20%**（1 / 5）                                                               |
| 空位填充（index floor） | **QQQ**                                                                        |
| 槓桿                    | **1.15×**（借入 15%）                                                          |
| 融資成本                | **3%／年**（按借入部分計）                                                     |
| 重新平衡                | **每月**一次                                                                   |
| 交易成本                | 每次成交 **15 bps**（0.15%，佣金 + 滑點）                                      |
| 執行時序                | 用收市價計信號，**下一個交易日**先成交（無前視）                               |

---

## 3. 每月決策流程

每個 rebalance 日（每月）：

```
步驟 1：定 universe
  → 攞「上一年年底」S&P 500 市值最大嘅 10 隻。
    （例：2015 年全年都用 2014 年底嗰 10 隻。因為 2015 年底個名單，2015 年初未知道。）

步驟 2：計動量
  → 對 universe 入面每隻：momentum = 收市價[今日] / 收市價[126 個交易日前] − 1

步驟 3：篩選 + 排名
  → 淨係保留 momentum > 0（升緊）嘅。
  → 由高到低排。

步驟 4：揀 Top 5
  → 取排名頭 5 隻。設實際揀到 n 隻（n ≤ 5）。

步驟 5：分配權重（未槓桿前，加總 = 100%）
  → 每隻揀中嘅股票： 20%（= 1 / 5）
  → 空位數 = 5 − n
  → QQQ 權重 += 空位數 × 20%
    （例：只得 3 隻正動量 → 3 隻各 20% = 60%，QQQ = 40%）
    （極端：全部跌、一隻都揀唔到 → QQQ = 100%）

步驟 6：加槓桿
  → 所有權重 × 1.15（總曝險 = 115%，借入 15%）
  → 借入部分每日扣融資成本 3% / 252

步驟 7：成交
  → 用「今日收市」算出嘅目標倉位，喺「下一個交易日」先落單調倉。
  → 每筆成交扣 15 bps。
```

---

## 4. 權重公式

設某月揀到 `n` 隻正動量股票 `s₁…sₙ`（n ≤ 5），槓桿 `L = 1.15`：

```
w(sᵢ)  = (1 / 5) × L              對每隻揀中嘅股票
w(QQQ) = ((5 − n) / 5) × L        空位用 QQQ 補
總曝險  = L = 115%（借入 15%）
```

每日淨回報：

```
r_組合 = L × r_未槓桿 − (L − 1) × (3% / 252)
       = 1.15 × r_未槓桿 − 0.15 × 0.0119%/日
```

## 5. Worked example（假設某月）

上一年年底 Top 10 = `AAPL MSFT NVDA AMZN GOOGL META AVGO BRK.B JPM XOM`

各自 126 日動量排名後，正動量得 4 隻：NVDA(+40%)、AVGO(+25%)、AMZN(+12%)、MSFT(+5%)，
其餘 6 隻 ≤ 0。

→ 揀中 4 隻，空位 = 1。未槓桿權重：

| 標的               | 權重 |
| ------------------ | ---- |
| NVDA               | 20%  |
| AVGO               | 20%  |
| AMZN               | 20%  |
| MSFT               | 20%  |
| QQQ（補 1 個空位） | 20%  |

→ × 1.15 槓桿後總曝險 115%（每項 × 1.15，借 15%）。
→ 用今日收市信號，下個交易日成交，扣 15 bps。

---

## 6. 點解係呢個組合（設計理由）

- **Top 5 而唔係 Top 1–2**：乾淨數據之下，集中（top1）係災難（CAGR ~5%）；分散到 5 隻 Sharpe 同回撤都好好多。
- **QQQ 托底**：個別股票唔夠強時，唔好揸現金（牛市現金拖後腿），跟返指數，令策略唔會大幅輸畀 QQQ。
- **1.15× 槓桿**：未槓桿版回撤 -30.6% < QQQ -35.1%，有「風險裕度」。加少少槓桿把裕度換成回報，risk 拉返同 QQQ 一樣 → 同風險、更高回報。
- **滯後 1 年揀 universe**：年底市值名單要年尾先知，唔可以年頭就用，否則係前視作弊。
- **每月 rebalance**：動量高換手，月度平衡壓低成本（日度會被成本食凸）。

## 8. 風險同局限（落注前必讀）

1. **-34.6% 回撤係真會發生，而且係槓桿後**——跌市有 margin 壓力，要頂得住先好做。
2. **Edge 好薄**：個贏主要係「同 QQQ 差唔多回報但跌少啲，再用槓桿放大」，唔係穩賺神技。若未來 QQQ 嘅回撤行為改變，個裕度會縮。
3. **逐年只 7/16 贏 QQQ**：全期嘅贏靠少數大年份（如 2024）撐起，return edge 喺噪音內；穩健嗰 part 係 drawdown 優勢。
4. **仲係 backtest**：真錢未行過。實盤要細注先跑一排，確認成交、滑點、融資利率同假設夾到。
5. **融資利率風險**：3% 係估算；高息環境（2023+）retail margin 可達 5-6%。×1.15 之下影響細（5% 只蝕 ~0.3pp），但槓桿越高越敏感。
6. **數據局限**：S&P 500 Top 10 市值名單來自年度快照；真正再嚴謹要季度 point-in-time 數據（Norgate/CRSP）。
7. **唔係 Sharpe alpha，係 leverage + 減回撤（walk-forward 驗證）**：對 vol-target 版做過 36 組參數 grid + walk-forward。結論：Sharpe ≈ QQQ（0.87–0.97 vs 0.96，只 7/36 組贏），CAGR 嘅「贏」係 **leverage（多冒風險）**，唔係 risk-adjusted alpha；唯一 robust 嘅好處係**回撤細啲**（~-30% vs -35%）。而且**自適應揀 vol-target 參數會 overfit**（OOS Sharpe 跌到 0.82），所以**固定參數、唔好優化**。落注理由應該係「想要 QQQ 級回報但跌少啲、並接受槓桿放大風險」，唔係「搵到 alpha」。

## 9. 變體：無槓桿闊池集中版（raw CAGR 跑贏 QQQ）

上面個主策略（窄池 mega-cap × 1.15）靠 **槓桿 + 減回撤** 贏。如果**唔想用槓桿**，數學上唯一仲可以贏 **raw CAGR** 嘅辦法 = **集中**（concentration）。

### 9.1 同主策略嘅分別

| 項目     | 主策略（§1–8）         | 無槓桿集中版                  |
| -------- | ---------------------- | ----------------------------- |
| 股票池   | 每年 S&P 500 市值 Top10 | **S&P 500 全體**（breadth 係關鍵） |
| 持倉     | top5                   | top **N**（N = 3 / 5 / 10）   |
| 槓桿     | 1.15×                  | **冇（1.0×）**                |
| 贏嘅來源 | 槓桿 + 回撤裕度        | **闊池集中 → 高 beta／raw CAGR** |

其餘（126 日動量、正動量篩選、等權、QQQ 托底空位、月度、收市信號下一日成交、15 bps）**完全一樣**。

### 9.2 universe 規則（關鍵）

- 用 **S&P 500 全體成員**，唔係 mega-cap top10。
- 用每隻嘅 **"Date added"**（加入指數日期）做 point-in-time 資格：某月 rebalance 只可以揀**嗰陣已經係成員**嘅股票，避免 membership 前視。
- **breadth 係 edge 來源**：窄池（cap-weighted top10）嘅 top5 ≈ QQQ 重磅股，所以打和；闊池 top5 = 揀**全市場**最強 5 隻，集中度遠高過 QQQ → CAGR 先跑贏。

### 9.3 已驗證表現（2010+，含 15 bps 成本，兩個獨立 S&P 500 list）

官方 datasets list（503 隻）：

| 配置（無槓桿,+QQQ floor） | CAGR      | Sharpe | 最大回撤   | OOS16 CAGR |
| ------------------------- | --------- | ------ | ---------- | ---------- |
| top3                      | **31.6%** | 0.93   | **-52.4%** | 34.9%      |
| top5                      | 25.7%     | 0.89   | -41.7%     | 26.4%      |
| **top10（最務實）**       | 23.9%     | 0.94   | -36.9%     | 24.8%      |
| 長揸 QQQ                  | 19.3%     | 0.96   | -35.1%     | 20.4%      |

兩個獨立 list（hanshof + datasets）top3/5/10 **全部贏 QQQ CAGR（full + OOS）** → 集中 beat **robust**。

### 9.4 風險同局限（同主策略 §8 之外，額外）

1. **Sharpe ≈ 或低過 QQQ（0.89–0.94 vs 0.96）**：CAGR 贏完全係**承受更深回撤**換返嚟，唔係 risk-adjusted alpha。集中 = 無槓桿版嘅「多冒風險換回報」，同槓桿本質一樣，只係換咗形式。**冇免費 CAGR。**
2. **回撤隨集中度爆升**：top10 ≈ -37%（仲頂得順），top3 去到 **-52%**。揀 N 就係揀你頂唔頂得順幾深嘅坑。**top10 最划算**（贏 4.6pp、Sharpe 0.94 ≈ QQQ、回撤只深 ~2pp）。
3. **Survivorship caveat（未解決）**：兩個 list 都係**今日**嘅 S&P 500 成員，缺咗歷史上已退市嘅。相對 beat（集中 > 指數）OOS + walk-forward 企得住，但**絕對 CAGR 偏高**。要 Norgate / CRSP 退市數據行 `pit_backtest_momentum_rotation.py` 先 confirm 剩幾多 —— **呢個係唯一未做嘅驗證**。
