## 市況判斷

### 道指

- 用價格加權計算，唔係市值加權。
- 台灣、日本主要指數都係用價格加權計算。
- GS 對道指影響大，因為價格加權。

### VIX

| VIX 水平 | 解讀            |
| -------- | --------------- |
| `< 13`   | 沽貨            |
| `< 20`   | 正常市況        |
| `2X`     | 波動性強        |
| `30+`    | 買股，長遠會贏  |
| `60`     | 極端            |
| `80`     | 新冠 / 金融海嘯 |

## 選股同估值框架

### 估值方法

| 方法       | 適用類型                       | 例子                     |
| ---------- | ------------------------------ | ------------------------ |
| P/E        | 成熟盈利股、平台股、金融股     | AAPL / GOOG / MSFT / JPM |
| P/S        | 高增長、盈利未完全反映嘅公司   | COIN / NBIS              |
| P/OCF      | 現金流比會計盈利更重要嘅公司   | AMZN                     |
| ROTE / PTB | 銀行同投行，睇資本回報同帳面值 | JPM                      |

### 操作邏輯

- 高質股唔一定任何價都買，要等估值跌到低位區間。
- 銷售或人氣唔受歡迎嗰年，可以留意高質消費 / 科技股。
- 太受歡迎嗰年，反而要小心估值過熱。
- 美股強勢股可以用價內 Long Call，但要跟估值同回調幅度做。

## 板塊同主題

### 大型平台 / 廣告 / 雲

**相關股份：** AAPL、AMZN、GOOG、META、MSFT

**重點：**

- 平台股要睇核心業務護城河同新成長曲線。
- 雲業務、AI 算力、廣告效率係主要增長來源。
- P/E 收縮風險通常來自市場覺得增長曲線放慢。

### AI 基建 / 半導體

**相關股份：** AVGO、NVDA、TSM、NBIS、SOLS

**重點：**

- 半導體有周期，週期盡頭先係較佳入貨位。
- 設計、代工、AI 雲平台係唔同模式，唔好混埋一齊估值。
- 資本開支同折舊對代工製造商影響大。

**晶片公司模式：**

| 模式     | 公司        |
| -------- | ----------- |
| 純設計   | NVDA、AMD   |
| 代工製造 | TSM、中芯   |
| 綜合模式 | 三星、Intel |

### 金融 / 支付 / 交易

**相關股份：** AXP、JPM、GS、V / MA、COIN

**重點：**

- 銀行同投行要睇 ROTE、PTB、壞賬同做市收入。
- 支付網絡要留意穩定幣對護城河嘅衝擊。
- COIN 除咗 Crypto，亦想做「萬物交易所」。

### 醫療

**相關股份：** AMGN、MRK、XLV

**重點：**

- 藥業股唔易做，可以用 XLV 代替單一藥廠。
- 專利到期同新藥管線係估值核心。

### 工業 / 軍工

**相關股份：** BA、CAT、HON、GE

**重點：**

- BA 偏軍工 + 盈利復甦。
- CAT 跟全球經濟景氣，亦有發電渦輪機幫助 AI 機構。
- HON 有軍工同量子電腦部門。

**重點：**

- 市值最高投行。
- 同美國政府最親密。
- 代表美國國運。
- 金融 + 科技。

**業務：**

| 部門           | ROTE  |
| -------------- | ----- |
| 零售及社區銀行 | `32%` |
| 商業及投資銀行 | `18%` |
| 資產及財富管理 | `34%` |

**估值 / 會計：**

- ROTE = ROE 扣除商譽。
- PTB = PB 扣除商譽。
- 淨利息收入：`50%`；非利息收入：`50%`。
- PTB-ROTE 橫向對比：ROTE 同 PTB 成正比。
- `7-8%` ROTE 對應 `1` 倍 PTB。
- `22%` ROTE 對應 `3` 倍 PTB。
- 自營盤比例少；做市收入比例多。

## ETF 策略

### 核心原則

- 長線揸。
- 睇行業前景 / 揀時令板塊。
- 當大市大跌幅可以介入。
- 逢大跌做深入價內 Long Call（有錢人玩法）。
- 做 Long Call 唔好崇尚 `100%` 勝率，最重要係交易曲線。
- Feel 到唔對路就要捨得走，Feel 到對路就由得佢。
- Long Call 贏五倍主要靠 MU、SNDK、AMD。

**ETF 分類：**

1. 指數 ETF
2. 行業 ETF

---

### 指數 ETF

#### QQQM

**核心操作：**

- 最高位回 `15%` 做第一口 Long Call。
- 再跌 `15%` 再做一口 Long Call。
- 做價內 `15%` Long Call，做 `9個月` 至 `1年`。

**調整幅度參考：**

| 類型     | 跌幅  |
| -------- | ----- |
| 中期調整 | `15%` |
| 大型調整 | `30%` |

**規律：** 每一年都有一次重大調整浪，大約回 `15-30%`（唔計金融海嘯）。

#### SPYM

- 高位回 `5%` 就供一口。

#### TQQQ

- ❌ 唔可以長 hold。
- 有價格損耗；3 倍槓桿 ETF 會有重置損耗。
- 如果長期磨損，`1800` 日會變零。
- 絕大多數槓桿 ETF 唔可以長揸。
- TQQQ 不斷破頂，係因為佢係最強指數。
- 每 QQQ 跌 `15%` 做第一口正股。
- 最多 hold `9個月` 到 `1年`。
- 一係用極少金額買咗，唔好睇佢。

#### SQQQ

- ❌ 唔可以長 hold。
